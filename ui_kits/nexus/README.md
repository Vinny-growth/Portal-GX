# GX Capital · Nexus UI Kit

Recreations of core surfaces from the **GX Capital Nexus AI** product (Vinny-growth/gx-capital-nexus-ai).

## Files
- `index.html` — interactive click-thru: login → dashboard → CRM → mesa de operações
- `Sidebar.jsx` — navy-gradient sidebar with collapsible groups
- `PageHeader.jsx` — brutalist dark hero with GXC watermark
- `KpiTile.jsx` — dark/light KPI variants
- `Card.jsx` — sharp card with tech-bracket hover
- `Button.jsx` — primary / secondary / outline / ghost / dark
- `Tabs.jsx` — brutalist underline tabs
- `QuickActions.jsx` — gold panel with action grid
- `ActivityList.jsx` — recent activity rows
- `LoginScreen.jsx` — auth surface
